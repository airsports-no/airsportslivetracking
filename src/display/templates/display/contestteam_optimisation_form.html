Contestant scheduling

Scheduling the contestants should ideally be done before the task starts when there are no existing contestants in the task. Scheduling requires that there are no contestants from the first tracking start time to the latest finished by time when creating a schedule and will warn accordingly if this is the case to give you the opportunity to clean out the interval or change the initial time. The system first uses a simple algorithm to create a valid good schedule that ensures that no aircraft overtakes another while respecting the constraints below. It then optionally applies a linear programming (LP) optimiser to minimise the latest finish point time (to make the final contestant complete the task as early as possible).
