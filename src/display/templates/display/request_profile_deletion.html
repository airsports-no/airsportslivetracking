When clicking the below button you are requesting your user data to be deleted.

This page requires you to be logged in. If you're accessing the page through the app you are already logged in, otherwise you will be prompted to login. After logging in the profile deletion request will be sent and you will be returned to the landing page.

{% endif %} Requests to delete user data are reviewed manually before the following actions are performed:
  • If possible, your user profile and all associated data will be deleted
  • If you are registered as the crew in a team for any contest your profile cannot be deleted since this might interfere with the ranking of existing competitions. In this case your profile is anonymized by removing all identifying information from the profile. Flown tracks will remain in the system, but there is no way to tie them to a specific user.

Note that deleting a user profile from within the app will probably break the functionality of the app until it is restarted and you log in again with a new profile.
